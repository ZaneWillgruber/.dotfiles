# Dotfiles

My dotfiles for my Arch and Hyperland system

## Requirements

Make sure you have the following requirements on your system

### Git
```
pacman -S git
```

### Stow
```
pacman -S stow
```

## Installation
Clone repo

```
git clone https://github.com/ZaneWillgruber/.dotfiles.git
cd .dotfiles
```

Use stow to create symlinks
```
stow .
```
